## 美术设计方案概述

### 1. 视觉风格定位
- **主题**：太空科幻主题（延续现有贪吃蛇风格）
- **核心元素**：深邃背景、霓虹发光、科技感UI
- **色彩系统**：
  - 背景：`#0B0B2A`（深空蓝）
  - 强调色：`#7AFFAF`（霓虹绿）、`#00D4FF`（霓虹蓝）
  - 7种方块颜色：青色、金色、紫色、橙色、蓝色、绿色、红色

### 2. 界面布局设计

#### 整体结构
```
顶部信息栏：得分、等级、暂停按钮
主游戏区：10×20网格（200×400px）
预览区：显示下一个方块（100×100px）
控制按钮区：左移、旋转、右移、加速下落
```

#### 关键尺寸
- **GRID_SIZE**：20px
- **游戏板**：10列 × 20行
- **按钮尺寸**：50×50px
- **圆角半径**：10px

### 3. 核心设计特性

#### 3.1 方块渲染
- 每个方块18×18px（留2px间隙）
- 圆角2px
- 渐变填充 + 霓虹发光边框
- 高光效果（左上角）

#### 3.2 UI元素
- **按钮**：渐变背景 + 圆角 + 发光效果
- **文字**：霓虹发光效果
- **弹窗**：半透明背景 + 霓虹边框

#### 3.3 动画效果
- **行消除**：闪烁 + 淡出 + 下落填补（500ms）
- **方块放置**：发光增强（200ms）
- **背景**：星星闪烁动画

### B. Canvas绘制函数示例
```javascript
function drawBlock(x, y, color, glowColor) {
    ctx.shadowColor = glowColor;
    ctx.shadowBlur = 10;
    
    const gradient = ctx.createLinearGradient(x, y, x + GRID_SIZE, y + GRID_SIZE);
    gradient.addColorStop(0, color);
    gradient.addColorStop(1, adjustColor(color, -20));
    
    ctx.fillStyle = gradient;
    ctx.beginPath();
    ctx.roundRect(x + 1, y + 1, GRID_SIZE - 2, GRID_SIZE - 2, 2);
    ctx.fill();
    
    ctx.shadowBlur = 0;
    ctx.fillStyle = 'rgba(255, 255, 255, 0.3)';
    ctx.beginPath();
    ctx.roundRect(x + 2, y + 2, GRID_SIZE - 4, (GRID_SIZE - 4) / 2, 1);
    ctx.fill();
}
```

### C. 按钮绘制函数示例
```javascript
function drawButton(btn, text, isPressed = false) {
    const gradient = ctx.createLinearGradient(btn.x, btn.y, btn.x, btn.y + btn.h);
    gradient.addColorStop(0, isPressed ? '#1A1A3A' : '#2A2A4A');
    gradient.addColorStop(1, isPressed ? '#0A0A2A' : '#1A1A3A');
    
    ctx.fillStyle = gradient;
    ctx.strokeStyle = 'rgba(122, 255, 175, 0.3)';
    ctx.lineWidth = 1;
    
    ctx.beginPath();
    ctx.roundRect(btn.x, btn.y, btn.w, btn.h, 10);
    ctx.fill();
    ctx.stroke();
    
    ctx.fillStyle = '#7AFFAF';
    ctx.font = '20px Arial';
    ctx.textAlign = 'center';
    ctx.textBaseline = 'middle';
    ctx.fillText(text, btn.x + btn.w/2, btn.y + btn.h/2);
}
```
