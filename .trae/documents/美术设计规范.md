# 俄罗斯方块游戏 - 美术设计规范

## 1. 整体视觉风格

### 1.1 主题定位
- **风格**：太空科幻主题
- **基调**：深邃、科技感、霓虹发光
- **参考**：保留现有贪吃蛇的太空站风格，适配俄罗斯方块特性

### 1.2 色彩方案

#### 主色调
- **背景色**：`#0B0B2A`（深空蓝）
- **游戏区域背景**：`#1A1A3A`（深紫蓝）
- **边框颜色**：`#2A2A4A`（科技灰）

#### 强调色
- **主强调色**：`#7AFFAF`（霓虹绿）- 用于分数、按钮高亮
- **次要强调色**：`#00D4FF`（霓虹蓝）- 用于UI元素
- **警告色**：`#FF4757`（霓虹红）- 用于游戏结束提示

#### 方块颜色（7种俄罗斯方块）
| 方块类型 | 颜色 | 霓虹发光色 | 用途 |
|---------|------|-----------|------|
| I（长条） | `#00F0FF` | `#00FFFF` | 青色 |
| O（方形） | `#FFD700` | `#FFFF00` | 金色 |
| T（T形） | `#A855F7` | `#C084FC` | 紫色 |
| L（L形） | `#FF8C00` | `#FFA500` | 橙色 |
| J（J形） | `#4169E1` | `#6495ED` | 蓝色 |
| S（S形） | `#32CD32` | `#00FF00` | 绿色 |
| Z（Z形） | `#FF4757` | `#FF6B7A` | 红色 |

## 3. UI元素设计

### 3.1 方块设计

#### 基础方块
- **尺寸**：18×18px（留2px间隙）
- **圆角**：2px
- **效果**：
  - 内部渐变（从中心到边缘）
  - 霓虹发光外边框（2px）
  - 高光效果（左上角）

#### 方块渲染示例
```javascript
// 方块绘制函数
function drawBlock(x, y, color, glowColor) {
    // 发光效果
    ctx.shadowColor = glowColor;
    ctx.shadowBlur = 10;
    
    // 渐变填充
    const gradient = ctx.createLinearGradient(x, y, x + GRID_SIZE, y + GRID_SIZE);
    gradient.addColorStop(0, color);
    gradient.addColorStop(1, adjustColor(color, -20));
    
    ctx.fillStyle = gradient;
    
    // 圆角矩形
    ctx.beginPath();
    ctx.roundRect(x + 1, y + 1, GRID_SIZE - 2, GRID_SIZE - 2, 2);
    ctx.fill();
    
    // 高光
    ctx.shadowBlur = 0;
    ctx.fillStyle = 'rgba(255, 255, 255, 0.3)';
    ctx.beginPath();
    ctx.roundRect(x + 2, y + 2, GRID_SIZE - 4, (GRID_SIZE - 4) / 2, 1);
    ctx.fill();
}
```

## 4. 动画效果设计

### 4.1 方块下落动画
- **默认速度**：每300ms下落一格
- **加速下落**：每50ms下落一格
- **动画**：平滑移动，无闪烁

### 4.2 行消除动画
- **效果**：
  1. 消除行闪烁（白色）3次
  2. 消除行淡出（透明度从1到0）
  3. 上方方块下落填补空缺
- **持续时间**：500ms

### 4.3 方块放置动画
- **效果**：放置时方块发光增强
- **持续时间**：200ms

### 4.4 星星背景动画
- **效果**：星星亮度随机闪烁
- **更新频率**：每帧更新
- **实现**：使用正弦函数控制亮度

## 5. 特效设计

### 5.1 霓虹发光效果
```javascript
// 通用发光函数
function setGlow(color, blur = 10) {
    ctx.shadowColor = color;
    ctx.shadowBlur = blur;
}

function clearGlow() {
    ctx.shadowBlur = 0;
}
```

### 5.2 粒子效果（行消除）
- **粒子数量**：每格2-3个粒子
- **粒子颜色**：对应方块颜色
- **粒子运动**：向四周扩散，逐渐消失
- **持续时间**：300ms

### 5.3 震动效果（游戏结束）
- **效果**：游戏板震动
- **持续时间**：500ms
- **振幅**：5px

## 6. 响应式设计

### 6.1 屏幕适配
- **竖屏模式**：标准布局
- **横屏模式**：游戏板和预览区并排显示
- **小屏幕适配**：缩小按钮尺寸和间距

### 6.2 字体缩放
- 使用相对单位或根据屏幕宽度动态调整

## 7. 性能优化建议

### 7.1 渲染优化
- 使用离屏Canvas缓存静态元素（如网格线）
- 减少不必要的阴影效果
- 限制粒子数量

### 7.2 动画优化
- 使用requestAnimationFrame实现平滑动画
- 避免在动画循环中创建新对象

## 9. 与前端开发对接要点

### 9.1 Canvas绘制顺序
1. 背景层（深空蓝）
2. 星星层
3. 游戏板背景层
4. 网格线层
5. 已放置方块层
6. 当前方块层
7. 预览区层
8. UI层（分数、按钮等）

### 9.2 状态管理
- **游戏状态**：menu, playing, paused, gameover
- **方块状态**：falling, placed, clearing
- **动画状态**：使用标志位控制动画播放

### 9.3 事件处理
- 触摸事件：debounce处理，防止误触
- 键盘事件：支持方向键和空格键
- 暂停/继续：保存游戏状态

### 9.4 性能监控
- FPS监控：目标≥60fps
- 内存使用：避免内存泄漏
- 渲染时间：单帧<16ms
